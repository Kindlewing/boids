# Boids

<p align="center">
    A boids simulation (eventually) written in C, with minimal external dependencies.
</p>

## Getting Started

### Prerequisites

Linux:

- Cmake
- Make
- X11
- OpenGL

Windows:

- Visual Studio
- Cmake
- OpenGL

### Building

1. Clone the repository:

```bash
    git clone https://github.com/Kindlewing/boids
    cd boids
```

2. Generate Makefile (Linux) or Visual Studio Solution (Windows)
