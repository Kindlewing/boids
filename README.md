# Boids

<p align="center">
    A boids simulation (eventually) written in C, with minimal external dependencies.
</p>
## Getting Started

### Prerequisites

#### Linux

- X11
- OpenGL

#### Windows (not currently supported)

- OpenGL

### Building

1. Clone the repository:

```bash
    git clone https://github.com/Kindlewing/boids
    cd boids
```
